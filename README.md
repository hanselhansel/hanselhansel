# Hey, I'm Hansel

Mass-producing AI prototypes, one rabbit hole at a time. Currently deep in the rabbit hole of **spatial intelligence, world models, and embodied AI**.

I think the best way to learn AI is to ship with it. So that's what I do.

If you're thinking about how machines learn to see, move, and reason about the physical world, we should talk.

---

## What I'm building

- **[forbotsake](https://github.com/forbotsake/forbotsake)** — marketing skills for Claude Code. Auto-posts via Claude for Chrome, handles content generation across X, Substack, LinkedIn.
- **[ai-learning-engine](https://github.com/hanselhansel/ai-learning-engine)** — 32-feature adaptive learning engine with SM-2 spaced repetition, flex sessions, teach-back, and concept linking. Hub-and-spoke architecture, built on 40+ academic sources in cognitive science. Currently using it to study spatial AI. Works with Claude Code, OpenAI Codex, Cursor.
- **[context-cli](https://github.com/hanselhansel/context-cli)** — LLM readiness linter for websites. Scores content density, robots.txt, schema.org, and llms.txt on a 0-100 scale. Published on PyPI.
